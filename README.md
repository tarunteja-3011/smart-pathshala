# 🌱 Smart-Pathshala

**Project Proposal 2026** — *Tackling Cognitive Overload in Technical Education*  
*Prepared for Andhra Loyola Institute of Engineering and Technology*

Smart-Pathshala is a Gemini-powered cognitive learning agent that ingests engineering course syllabi, textbooks, and video lectures to provide personalized tutoring, visual gap analysis, and instant retrieval.

## 🚀 Live Demo
Access the live application here: [Smart-Pathshala Live](https://smart-pathshala-mmpawajtw7vgeq4wabactq.streamlit.app)

## ✨ Key Capabilities
* **Multimodal Syllabus Ingestion:** Upload PDF textbooks or lab manuals to query course material directly.
* **Contextual AI Peer:** Powered by Google's Gemini models with a calm, approachable "Soft Intelligence" persona.
* **YouTube Video Synthesis:** Index video lectures for instant jump-to-concept retrieval.
* **Visual Gap Analysis:** Identify missing prerequisites across B.Tech semesters.
